# 1689 Heritage Trail

An interactive historical atlas tracing the Particular Baptist heritage across England and Wales — from the emergence of the first congregations in the 1630s through the era of Spurgeon and the Metropolitan Tabernacle, centered on the **1689 Second London Baptist Confession of Faith**.

**Live Site:** [https://garridolecca.github.io/1689-heritage-trail/](https://garridolecca.github.io/1689-heritage-trail/)

## Four Eras

| Era | Period | Theme |
|-----|--------|-------|
| **Origins** | 1612–1644 | First churches and the 1644 Confession |
| **Persecution** | 1644–1689 | The Clarendon Code and secret confession-making |
| **Toleration** | 1689–1792 | Public confession and institutional growth |
| **Mission** | 1792–1892 | Global mission and the Prince of Preachers |

## Features

- Interactive map with 35+ historical locations across England and Wales
- Color-coded markers and routes by era
- Era filtering to isolate specific periods
- Timeline panel with 45 chronological events
- Key Figures panel with 16 biographical cards
- Click any location for detailed historical narratives
- Responsive design (mobile, tablet, desktop)

## Tech Stack

- **ArcGIS Maps SDK for JavaScript 4.31** — mapping and geospatial visualization
- **Calcite Design System** — UI components
- **Vanilla ES Modules** — zero-build, no dependencies
- **GitHub Pages** — static hosting
